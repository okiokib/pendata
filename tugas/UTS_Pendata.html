  1. Pemotongan Data atau Data Slicing

pisahkan data menjadi data training and data test. Kita dapat dengan mudah melakukan langkah ini menggunakan metode train_test_split() dari sklearn:

Variabel features_train berisi data fitur untuk training, features_test berisi data fitur untuk test, target_train berisi target untuk training, dan target_test berisi target untuk test.

features = df_copy.values[:,:13] 
target = df_copy.values[:,13]  
features_train, features_test, target_train, target_test = train_test_split(features,target, test_size = 0.33, random_state = 10)

  1. Implementasi Gaussian Naive Bayes

Setelah proses preprocessing data selesai dilakukan, langkah selanjutnya adalah menerapkan algoritma pembelajaran mesin atau machine learning. Pada kasus ini, kita akan menggunakan modul GaussianNB dari library sklearn. Dengan menggunakan modul ini, kita dapat membangun model yang mampu melakukan klasifikasi data dengan menggunakan pendekatan probabilitas dari distribusi Gaussian.

clf = GaussianNB()
clf.fit(features_train, target_train)
target_pred = clf.predict(features_test)

  1. Akurasi model Gaussian Naive Bayes

accuracy = accuracy_score(target_test, target_pred, normalize = True)

print(f"Akurasi model Naive Bayes: {accuracy:.2%}")

Akurasi model Naive Bayes: 79.00%

Kesimpulan#

Berdasarkan nilai akurasi sebesar 79%, dapat disimpulkan bahwa data yang digunakan untuk melatih model telah memberikan hasil yang baik. Untuk menguji performa model yang telah dilatih, kita dapat membuat dataset uji secara acak dan melakukan pengujian untuk melihat seberapa baik kinerja model Gaussian Naive Bayes yang telah dibangun. Hal ini dapat membantu kita untuk mengetahui seberapa baik model dapat digunakan untuk memprediksi klasifikasi data baru secara akurat.

Langkah terakhir setelah melakukan inisialisasi objek knn dari kelas KNeighborsRegressor yang merupakan algoritma K-Nearest Neighbors untuk melakukan regresi, adalah melakukan randomisasi indeks dari dataset menggunakan np.random.permutation() dengan seed 1 untuk memastikan hasil yang konsisten. Selanjutnya, dataset dibagi menjadi 75% data untuk training dan 25% untuk testing dengan menggunakan variabel train_index. Data training dan data testing disimpan di variabel training_data dan testing_data, masing-masing.

Setelah model dilatih, prediksi dilakukan pada data testing dengan memanggil method predict() pada objek knn dengan argumen input berupa fitur-fitur dari data testing yang ingin diprediksi.

Setelah itu, akurasi model dihitung dengan mengukur berapa persen data testing yang berhasil diprediksi dengan menggunakan threshold default 0.5.

Terakhir, hasil akurasi model dicetak.

knn = KNeighborsRegressor()

np.random.seed(1)
index_shuffled = np.random.permutation(df_copy.index)
shuffled_car_data = df_copy.loc[index_shuffled]

train_index = int(shuffled_car_data.shape[0] * 0.75)
training_data = shuffled_car_data.iloc[0:train_index]
testing_data = shuffled_car_data.iloc[train_index:]

knn.fit(training_data[['age', 'sex', 'cp', 'trestbps', 'chol', 'fbs', 'restecg', 'thalach', 'exang', 'oldpeak', 'slope', 'ca', 'thal']], training_data['target'])

prediction = knn.predict(testing_data[['age', 'sex', 'cp', 'trestbps', 'chol', 'fbs', 'restecg', 'thalach', 'exang', 'oldpeak', 'slope', 'ca', 'thal']])

accuracy = np.mean(np.abs(prediction - testing_data['target']) <= 0.5)

print(f"Akurasi model KNN: {accuracy:.2%}")

Akurasi model KNN: 92.11%
