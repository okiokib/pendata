  1. Memisahkan fitur dan target

# features and target

  1. Melakukan Normalisasi Data pada Fitur dan Target

# Normalisasi pada X

# Normalisasi pada y
label = LabelEncoder()

  1. Menyimpan data normalisasi

  • knn_pca = KNeighborsClassifier(n_neighbors=3): Membuat objek dari kelas KNeighborsClassifier dengan parameter n_neighbors=3, yang berarti jumlah tetangga terdekat yang akan digunakan dalam proses klasifikasi.

  • knn_pca.fit(X_train_new, y_train_new): Melatih model KNN dengan data latih yang telah diproses menggunakan PCA. X_train_new adalah matriks fitur dari data latih yang telah direduksi dimensinya dengan PCA, dan y_train_new adalah vektor target yang sesuai dengan data latih.

  • y_prediksi_knn_pca = knn_pca.predict(X_test_new): Memprediksi kelas target untuk data uji yang telah diproses menggunakan PCA. X_test_new adalah matriks fitur dari data uji yang telah direduksi dimensinya dengan PCA.

  • print(‘Accuracy: {:.2f}’.format(accuracy_score(y_test_new, y_prediksi_knn_pca))): Menghitung dan mencetak akurasi prediksi model KNN dengan PCA. accuracy_score adalah fungsi untuk menghitung akurasi berdasarkan perbandingan antara y_test_new (kelas target sebenarnya) dengan y_prediksi_knn_pca (kelas target yang diprediksi).

  • Akurasi merupakan metrik evaluasi yang mengukur sejauh mana prediksi model cocok dengan data sebenarnya. Outputnya akan mencetak nilai akurasi dalam format desimal dengan dua angka di belakang koma.

B. Membangun model KNN dengan PCA#

# Membangun PCA dengan 3 fitur
pca = PCA(n_components=3)
X_new=pca.fit_transform(X)

# Menyimpan pca ke file
with open('pca.pkl', 'wb') as file:
    pickle.dump(pca, file)

# Split data into train and test data
X_train_new, X_test_new, y_train_new, y_test_new = train_test_split(X_new, y, test_size = 0.2, random_state=42, stratify=y)

# Membangun model KNN dengan PCA
knn_pca = KNeighborsClassifier(n_neighbors=3)

# Melatih model dengan data latih
knn_pca.fit(X_train_new, y_train_new)

# Memprediksi kelas target untuk data uji
y_prediksi_knn_pca = knn_pca.predict(X_test_new)

# Menghitung akurasi prediksi
print('Accuracy: {:.2f}'.format(accuracy_score(y_test_new, y_prediksi_knn_pca)))

Accuracy: 0.97
